# Comandos

**1. Versão do Kernel**:
``` uname -r ``` </br> </br>
**2. GUI atual**:
``` env |grep CURRENT_DESKTOP ``` </br> </br>
**3. Usuário ADM**:
``` sudo su ```
**4. Criar diretório**:
``` mkdir ```
**5. Remover diretório**:
``` rmdir / rm -r ```
**6. Criar arquivo**:
``` touch ```
**7. Acessar arquivo*:
``` cat ```
