# Comandos

**1. Versão do Kernel**:
``` uname -r ``` </br> </br>
**2. GUI atual**:
``` env |grep CURRENT_DESKTOP ``` </br> </br>
**3. Usuário ADM**:
``` sudo su / su - ``` </br> </br>
**4. Criar diretório**: 
``` mkdir ``` </br> </br>
**5. Remover diretório**:
``` rmdir / rm -r ``` </br> </br>
**6. Criar arquivo**:
``` touch ``` </br> </br>
**7. Acessar arquivo**:
``` cat ``` </br> </br>
**8. Copiar arquivo/diretório**:
``` cp / cp -r (diretório)``` </br> </br>
**9. Mover ou Renomear arquivo/diretório**:
``` mv / mv -r (diretório) ``` </br> </br>
